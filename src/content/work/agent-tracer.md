AgentTracer brings the discipline of distributed tracing to AI agent workflows. It captures structured spans for each tool call, decision change, and execution branch, surfacing what an agent did and why. Built for debugging and evaluation under privacy constraints — metadata-only by default, never the raw prompts or responses.
